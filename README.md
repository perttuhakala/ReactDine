### Tools and software needed for the application
```
Visual Studio code
Nodejs
```

How to run locally (The app is currently only usable locally): 

### 1. Clone repository: 
$ git clone https://github.com/perttuhakala/ReactDine
   
### In Visual Studio code:

### 2. Run backend: 
```
$ cd ReactDine/server
$ node server.jsx
```
### 3. Run frontend:
```
$ cd project/client
$ npm install
$ npm run dev
```
